# 湖应商城
## 1. 项目介绍
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;“湖应商城”项目是一套前后端分离的电商系统，基于SpringBoot+MyBatis+Vue实现。 系统包含阿里云短信注册、登录、轮播图、商品推荐、分类推荐、商品搜索、商品展示、购物车、微信支付、个人中心、地址管理等模块。

## 3. 技术栈
### 3.1 前端技术栈
| ***\*技术\**** | ***\*说明\**** |
| -------------- | -------------- |
| Vue            | 前端框架       |
